# aeromaticdrone.com

Static marketing site for **Aeromatic Drone Services** — Astro, dark schematic theme, deployed via Cloudflare Pages.

## Quick start

```bash
cd ~/projects/aeromatic-site
source ~/.nvm/nvm.sh
npm install
npm run dev
```

## Docs

- [ASSETS.md](./ASSETS.md) — where to put photos and videos
- [DEPLOY.md](./DEPLOY.md) — GitHub + Cloudflare Pages setup

## Pages

| Path | Purpose |
|------|---------|
| `/` | Home |
| `/gallery` | Auto-indexed images from `public/gallery/` |
| `/blog` | Markdown posts in `src/content/blog/` |
| `/about` | Bio + Airomatic legacy |
| `/contact` | Form + map embed |
