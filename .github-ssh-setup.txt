Add this SSH public key to GitHub (one time):
https://github.com/settings/ssh/new

Paste the key from: ~/.ssh/id_ed25519.pub

Then push:
  cd ~/projects/aeromatic-site
  git push -u origin main

Or authenticate with GitHub CLI (uploads this key automatically):
  gh auth login -h github.com -p ssh -w
  git push -u origin main

Remote is set to: git@github.com:cantrell78-max/aeromatic-site.git
